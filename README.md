# IntermissionToID24

This is a tool for converting ZDoom intermission scripts into the new JSON-based interlevel format of Nightdive's 2024 Doom + Doom II remaster, written in Java.

## Resources
- [ZDoom Wiki - Intermission script](https://zdoom.org/wiki/Intermission_script)
- [ID24 specification - JSON Lump](https://docs.google.com/document/d/1SGWMFggsARYKWRsSHm_7BPMrGCEcm6t6DCpWiQyfjTY/edit)
- [ID24 specification - Interlevel Lumps](https://docs.google.com/document/d/1IVJ4nt5f20_NUrMrr9gl-UDStq1jexo5y4XTL1qd4vs/edit)
